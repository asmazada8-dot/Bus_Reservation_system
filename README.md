# Bus Reservation System in C

A simple console-based bus seat reservation system built in C using 2D arrays.

## Features
- Book and Cancel seats
- Separate seating for Male [1-10] and Female [11-20]
- 5 Rows x 4 Columns layout
- View seat chart with booked/available seats
- View total bookings and collection

## Ticket Price
- Male: Rs. 500 per ticket
- Female: Rs. 450 per ticket

## How to Run
1. Open the project in Dev-C++ or any C compiler
2. Compile: `gcc Bus_Reservation_System.c -o bus`
3. Run: `./bus`

## Technologies Used
- Language: C
- Concept: 2D Arrays, Functions
